# Timed Tab
A Chrome extension timer that automatically closes a tab after timer expires.

## The Problem It Solves
Ever find yourself spending too much time on distracting websites? Timed Tab helps you stay disciplined by automatically closing tabs after a set time limit. Perfect for:
- Limiting social media browsing
- Taking timed breaks from work
- Time-boxing research sessions
- Preventing endless scrolling

## Key Features
- Timer Input - Set duration in MM:SS format (e.g., 25:00)
- Audio Alert - Sound notification when timer completes
- Auto-Close - Automatically closes the tab when time expires
- Visual Countdown - Real-time display with red warning at 10 seconds
- Reset Option - Stop the timer early if needed

## Installation Instruction
1. Download or clone this repository to your computer
2. Open chrome and navigate to [chrome://extensions/](/chrome://extensions/)
3. Enable Developer mode using the toggle in the top right corner
4. Click Load unpacked and select the project folder
5. The Timed Tab extension will appear in your toolbar

## Usage
1. Click the Timed Tab extension icon in your Chrome toolbar
2. Enter your desired time in MM:SS format (e.g., 05:30 for 5 minutes 30 seconds)
3. Click Start to begin the countdown
4. The timer will count down and display the remaining time
5. When time expires:
    - An alarm sound will play
    - The current tab will automatically close
6. Click Reset at any time to cancel the timer
